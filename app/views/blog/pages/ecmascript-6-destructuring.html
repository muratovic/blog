Both interfaces provide traverse the collection and fetch objects from it. But, there are 2 differences between Iterator and Enumeration interface. 1. Iterator has remove() method which can delete the current node while traversing the collection. 
    
    public interface Iterator {
       public boolean hasNext();
       public Object next();
       public void remove();
    }
    

public interface Iterator {
       public boolean hasNext();
       public Object next();
       public void remove();
}
Enumeration interface doesnt have this method. Enumeration acts as Read-Only interface. 
public interface Enumeration {
       public boolean hasMoreElements();
       public Object nextElement();
}
      2. Iterator is more secure and safe compared with Enumeration interface. Iterator doesnt allow multiple threads modify the collection at the same time. Therefore, ConcurrentModificationException is not thrown while traversing the collection through the use of Iterator interface. Whereas Enumeration interface cannot provide this feature. Enumeration is not thread-safe interface.
