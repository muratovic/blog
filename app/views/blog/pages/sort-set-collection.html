If your set is not TreeSet implementation, you have the following options to sort your set collection.

For example, suppose that you have a HashSet collection as shown below;

        Set hashSet = new HashSet();
        hashSet.add("murat");
        hashSet.add("ayse");
        hashSet.add("oguz");

You can one of these methods to sort your set collection;

1. One method is to create a new TreeSet whose constructor has any collection type.

        TreeSet treeSet = new TreeSet(hashSet);
        for (final String element : treeSet) {
            System.out.println(element);
        }

2. Another option is to create a new ArrayList whose constructor has any collection type and sort this List via Collections.sort(List) method.

        List list = new ArrayList(hashSet);
        Collections.sort(list);

        for (final String element : list) {
            System.out.println(element);
        }

3. Other option is to convert this set into array through the use of toArray method and sort this array via Arrays.sort(Object[]) method.

        Object[] array = hashSet.toArray();
        Arrays.sort(array);

        for (final Object element : array) {
            System.out.println(element);
        }

All these options shown above will have the same output.
