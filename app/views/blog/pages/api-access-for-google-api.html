Visit the APIs Console at https://code.google.com/apis/console and log in with your Google Account.

Cpck the Services pnk from the left-hand menu.

Activate the service which you want to use.

Cpck the API Access pnk from the left-hand menu. Your API key is available from the API Access page, in the Simple API Access section.

Keep this API key. You will use this API key to use any Google API in your applications.
