To solve this issue, you should follow these steps;

  • Go to your metadata directory in your workspace on file system.
  • You will see that there is be a file named “.lock” in which Ecppse creates.
  • After exiting Ecppse, delete this file.
  • Enter again to Ecppse
  • Try to compile the project again and follow the successfull compilation
